# :small_blue_diamond: **Registros de Solicitações de Manutenção - IFMG**

Este projeto tem como objetivo analisar os dados de solicitações de manutenção (predial e informática) no **Instituto Federal de Minas Gerais (IFMG)** e prever a demanda de futuros chamados.

---

## 📋 **Objetivo do Projeto**

- Analisar dados de solicitações de manutenção.
- Identificar padrões e tendências nos chamados.
- Prever a demanda de futuras solicitações.

---

## 📊 **Fonte de Dados**

Os dados utilizados são públicos e foram coletados de **2019 a agosto de 2024**.

📁 **Link para acesso à base de dados**: [Clique aqui](https://dadosabertos.ifmg.edu.br/dataset/chamados-de-suporte)

---

## 🛠️ **Tecnologias e Ferramentas**

- **:computer: Ferramentas**: Python, Google Colab
- **:books: Bibliotecas**:
  - `pandas` – Manipulação e análise de dados
  - `numpy` – Operações numéricas e manipulação de arrays
  - `matplotlib` e `seaborn` – Criação de gráficos e visualizações
  - `prophet` – Modelagem e previsão de séries temporais
  - `scikit-learn` – Métricas de avaliação para regressão (`mean_absolute_error`, `mean_squared_error`)

# 🔹 **Maintenance Request Records - IFMG**

This project aims to analyze maintenance request data (building and IT) at the **Instituto Federal de Minas Gerais (IFMG)** and forecast demand for future requests.

---

## 📋 **Project Objective**

- Analyze maintenance request data.
- Identify patterns and trends in requests.
- Forecast future demand for requests.

---

## 📊 **Data Source**

Public data from **2019 to August 2024** were used.

📁 **Link to access the database**: [Click here](https://dadosabertos.ifmg.edu.br/dataset/chamados-de-suporte)

---

## 🛠️ **Technologies and Tools**

- **:computer: Tools**: Python, Google Colab
- **:books: Libraries**:
  - `pandas` – Data manipulation and analysis.
  - `numpy` – Numerical operations and array manipulation.
  - `matplotlib` and `seaborn` – Creating graphs and data visualizations.
  - `prophet` – Time series modeling and forecasting.
  - `scikit-learn` – Regression evaluation metrics (`mean_absolute_error`, `mean_squared_error`).
